3단계로 시작하는 AI 코딩 어시스턴트

  1. VS Code 마켓플레이스에서 Jupyter Gemini Assistant를 검색하고 설치하세요.
  2. VS Code에서 Jupyter 노트북을 열고 평소처럼 코딩을 시작하세요.
  3. 오류가 발생하면 코드 셀 옆의 '분석' 버튼을 클릭하여 AI의 도움을 받으세요.
